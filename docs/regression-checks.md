# 반복 오류 회귀 체크리스트

이 파일은 작업 중 반복적으로 등장한 오류를 기록합니다.
`/bug_check` 커맨드를 실행하면 Playwright MCP가 각 항목을 자동으로 점검합니다.

## 항목 형식

```
## BUG-XXX: 오류 제목

**현상**: 무슨 일이 생겼는가
**재현 조건**: 어떤 경로/상황에서 발생하는가
**확인 방법**: Playwright가 어떤 동작을 해서 어떤 결과를 봐야 통과인가
**등록일**: YYYY-MM-DD
**상태**: 활성 | 비활성
```

## BUG-001: PC/모바일 환경 판별 오류

**현상**: PC 환경에서 브라우저 너비가 줄어들면 모바일 레이아웃·플로우가 잘못 적용됨.
반응형 너비(px) 기준이 아니라 실제 기기 유형(User-Agent, 포인터 방식 등)을 기준으로 판별해야 함.

- PC: 너비에 관계없이 항상 PC 플로우 유지 (케이스 상세 → AI 차팅 우선 표시)
- 모바일: 실제 모바일 기기일 때만 모바일 플로우 적용 (케이스 상세 → 문진 입력 우선 표시)

**재현 조건**: PC 브라우저에서 창 너비를 모바일 수준(예: 390px)으로 줄인 뒤 케이스 상세 페이지 접근.
모바일 레이아웃(문진 입력 영역이 위, AI 차팅이 버튼 뒤)이 적용되면 오류.

**확인 방법**:

1. Playwright에서 PC User-Agent + 좁은 뷰포트(390px)로 `/cases/[id]` 접속
   → AI 차팅 섹션이 문진 입력 영역보다 위에 렌더링되면 통과
2. Playwright에서 모바일 에뮬레이션(iPhone 등) + 동일 URL 접속
   → 문진 입력 영역이 AI 차팅보다 위에 렌더링되면 통과

**등록일**: 2026-04-24
**상태**: 활성
