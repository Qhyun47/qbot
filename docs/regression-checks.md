# 반복 오류 회귀 체크리스트

이 파일은 작업 중 반복적으로 등장한 오류를 기록합니다.
`/bug_check` 커맨드를 실행하면 Playwright MCP가 각 항목을 자동으로 점검합니다.

## 항목 형식

```
## BUG-XXX: 오류 제목

**현상**: 무슨 일이 생겼는가
**재현 조건**: 어떤 경로/상황에서 발생하는가
**확인 방법**: Playwright가 어떤 동작을 해서 어떤 결과를 봐야 통과인가
**등록일**: YYYY-MM-DD
**상태**: 활성 | 비활성
```
