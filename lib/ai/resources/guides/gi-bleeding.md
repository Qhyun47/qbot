# GI Bleeding 문진 가이드

> **C.C. 변형**: Hematemesis(토혈) / Melena(흑변) / Hematochezia(혈변)도 동일 가이드 사용

## 핵심 문진 항목

### 출혈 특성

- [ ] **Type**: Hematemesis(토혈) / Melena(흑변) / Hematochezia(혈변)?
  - 둘 이상 동시 발생 가능 → 각각 확인
- [ ] **Hematemesis**: 횟수? 양(컵/소량)? 발생 시점?
- [ ] **Melena**: 횟수? 양? 발생 시점?
- [ ] **Hematochezia**: 선홍색 vs 암적색? 대변에 묻음 vs 섞임?
- [ ] **Onset**: 언제 처음 시작됐는가?
- [ ] **Trigger**: 직전 음식 섭취? 음주? 구역/구토 후 발생?

### 혈역학적 안정성 평가

- [ ] Dizziness / Presyncope / Syncope 있는가?
- [ ] 기립 시 어지러움?

### 위장관 관련 이력

- [ ] 이전 EGD(위내시경) / Colonoscopy 이력 및 결과
- [ ] 이전 유사 출혈 에피소드

### NPO 상태

- [ ] 마지막 고형식 섭취 시점
- [ ] 마지막 수분 섭취 시점

### 동반 증상

- [ ] Abdominal pain
- [ ] N/V
- [ ] Dyspnea
- [ ] Fever

### 과거력 및 위험인자

- [ ] 간경변 / 간질환 (정맥류 출혈 위험)
- [ ] 소화성 궤양 이력
- [ ] 음주력
- [ ] NSAID / Aspirin / 항응고제 / 항혈소판제 복용 여부

## 감별 진단 포인트

**Upper GI bleeding (UGIB)** — 주로 Hematemesis + Melena
| 진단 | 주요 특징 |
|------|-----------|
| **Peptic ulcer** | 가장 흔한 원인, NSAID/Aspirin 복용, epigastric pain 이력 |
| **Esophageal varices** | 간경변 + 대량 출혈 + hematemesis |
| **Mallory-Weiss tear** | 심한 구역/구토 후 발생 |
| **Gastric cancer** | 고령 + 체중감소 + 만성 증상 |

**Lower GI bleeding (LGIB)** — 주로 Hematochezia
| 진단 | 주요 특징 |
|------|-----------|
| **Diverticulosis** | 고령 + painless hematochezia + 갑작스러운 발생 |
| **Colorectal cancer** | 만성 혈변 + weight loss + 배변 습관 변화 |
| **IBD** | 젊은 연령 + 만성 설사 + 복통 동반 |
| **Hemorrhoid** | 배변 시 선홍색 출혈, 통증 동반 가능 |

---

## ⚠️ 위험 신호 (즉시 확인)

- **Tachycardia + hypotension** → 혈역학적 불안정 → 즉각 IV access + 수액 소생
- **Syncope / Presyncope** → 급성 대량 출혈
- **대량 hematemesis** (컵 이상) → 기도 흡인 위험
- **Known 간경변 + hematemesis** → Variceal bleeding (사망률 높음, 즉각 조치 필요)
- **항응고제 복용 중 대량 출혈** → 약물 역전(reversal) 고려
