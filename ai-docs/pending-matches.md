# Pending Matches

이 파일은 정형 C.C.로 등록되었으나 아직 가이드라인 또는 상용구와 매칭되지 않은 항목을 추적합니다.
Claude는 가이드라인 또는 상용구를 추가할 때 반드시 이 파일을 읽고, 새로 추가하는 리소스와 매칭 가능한 항목이 있으면 사용자에게 매칭 여부를 묻습니다.

---

## 미매칭 항목

(현재 없음)

## 매칭 완료 기록

| ---- | ------------- | --------- | ------ |

---

## 항목 추가 양식

C.C.를 먼저 등록하고 가이드라인/상용구가 아직 없는 경우, 아래 양식으로 항목을 추가합니다.

```
### {C.C. 이름}
- 등록일: YYYY-MM-DD
- 가이드라인: 없음 (또는 키 이름)
- 상용구: 없음 (또는 키 이름)
- 메모: -
```
