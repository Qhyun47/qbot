# Pending Connections

이 파일은 C.C. / 가이드라인 / 상용구 간의 **대기 중인 커넥션**을 추적합니다.

리소스 등록 시 연결하려는 상대 리소스가 아직 존재하지 않을 경우, 아래 테이블에 기록합니다.
이후 해당 리소스가 등록될 때 이 파일을 확인하여 연결 여부를 사용자에게 질문합니다.

---

## 사용 규칙 요약

- **C.C. 등록 시**: 언급된 guideKey가 guide-list.json에 없으면 섹션 1에 기록. templateKey가 template-list.json에 없으면 섹션 2에 기록. 섹션 3/4에서 이 C.C.를 기다리는 항목 확인 후 연결 여부 질문.
- **가이드라인 등록 시**: 언급된 C.C.가 cc-list.json에 없으면 섹션 3에 기록. 섹션 1에서 이 guideKey를 기다리는 C.C. 확인 후 연결 여부 질문.
- **상용구 등록 시**: 언급된 C.C.가 cc-list.json에 없으면 섹션 4에 기록. 섹션 2에서 이 templateKey를 기다리는 C.C. 확인 후 연결 여부 질문.
- **수락 시**: cc-list.json 업데이트 + 해당 행 삭제
- **거절 시**: 해당 행 삭제만 (재연결은 수동)
- **중복 방지**: 기록 전 동일 (C.C., key) 쌍이 이미 존재하는지 확인
