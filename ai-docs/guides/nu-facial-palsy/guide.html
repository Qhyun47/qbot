< NU – Facial palsy >

기본 lab + Brain routine CT

   * Facial (Upper/Lower) (motor/sensory) check!

    - 이마주름 / 눈감기 / 볼 부풀리기 / 혀 내밀기 / 꽉 깨물기 등

    - lower만 이상 있을 시, CODE RED time 확인하기

    - Upper 까지 palsy 있는 bell's palsy 의심 환자는 코레X!!!!

    - both limb motor/sensory 확인


   * History taking

     - underlying disease : DM(-), 대상포진(-)

     - weight, NPO time

     - recent URI Sx.(-)

     - no vesicle(-)

     - taste change(-), dry mouth(-), No tears(-)

     - auricle pain(-), retromastoid pain(-), tenderenss(-), tinnitus(-), hearing loss(-)


   * Ramsay-Hunt Syndrome? (→ OL notify)

     - Painfun rash

     - Hearing loss
     - Facial paralysis


* Peripheral type은 code red 띄우면 안되겠죠?

Central type : 증상이 입 주변에만!

     ex ) 오른쪽 이마 주름 잡히지만 입꼬리 안올라감

Peripheral type = Bell's palsy : 얼굴 위쪽 증상 + 입 주변 증상

     ex ) 오른쪽 이마 주름 안잡히고 입꼬리도 안올라감
