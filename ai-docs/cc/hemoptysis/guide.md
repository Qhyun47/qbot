# Hemoptysis 문진 가이드

## 핵심 문진 항목

### 출혈 특성

- [ ] **Onset**: 언제부터? 갑자기 vs 서서히?
- [ ] **Color**: 선홍색(bright red) vs 암적색(dark red) vs 가래에 섞임?
- [ ] **Amount**: 총량은? (teaspoon / 반 컵 / 컵 이상)
- [ ] **Count**: 몇 차례?
- [ ] **Trigger**: 기침 중 발생? 자연 발생? 구역 후?

### 동반 증상

- [ ] Dyspnea
- [ ] Chest pain (pleuritic?)
- [ ] Fever / Chill
- [ ] Cough (만성 기침 여부)
- [ ] Night sweats
- [ ] Recent weight loss
- [ ] Dizziness (대량 출혈 시)

### 결핵 관련

- [ ] 과거 폐결핵 치료 이력
- [ ] 최근 결핵 환자 접촉력
- [ ] BCG 접종 여부 (소아)

### 과거력

- [ ] 기관지확장증, 폐암, 기관지염, 폐렴 이력
- [ ] 흡연력 (갑년)
- [ ] 항응고제 / 항혈소판제 복용 여부
- [ ] 이전 유사 에피소드

---

## 감별 진단 포인트

| 진단                        | 주요 특징                                     |
| --------------------------- | --------------------------------------------- |
| **기관지염 / 기관지확장증** | 만성 기침 + 소량 각혈, 가장 흔한 원인         |
| **결핵 (TB)**               | Night sweats, weight loss, 상엽 병변, 접촉력  |
| **폐암**                    | 고령 + 흡연력 + weight loss + 만성 기침       |
| **Pulmonary embolism**      | Pleuritic chest pain + dyspnea + leg swelling |
| **Pneumonia**               | Fever + productive cough + 급성 발병          |
| **항응고제 관련**           | 복용 약물 확인                                |

---

## ⚠️ 위험 신호 (즉시 확인)

- **Massive hemoptysis** (>200mL/24h 또는 컵 이상) → 기도 확보 우선
- **SpO2 감소 동반**
- **Weight loss + night sweats** → TB / 폐암 의심
- **Known 폐암 + 갑작스러운 대량 각혈** → 혈관 침범 가능
- **항응고제 복용 중** → 약물 과용량 가능성
