# Dyspnea 문진 가이드

## 핵심 문진 항목

### 발생 양상

- [ ] **Onset**: 언제부터? 갑자기(acute) vs 서서히(gradual)?
- [ ] **Orthopnea**: 누우면 숨이 더 찬가? (베개 몇 개?)
- [ ] **PND** (Paroxysmal nocturnal dyspnea): 자다가 숨차서 깬 적 있는가?
- [ ] **Nocturnal symptom**: 야간에 증상 악화?

### 악화/완화 요인

- [ ] **Aggravating factor**: 운동 시? 눕을 때? 투석 후 체중 증가?
- [ ] **Relieving factor**: 앉으면 나아지는가? 기관지확장제 효과?
- [ ] **Exercise tolerance**: 평소와 비교해 활동 능력 감소 정도?

### 동반 증상

- [ ] Cough / Sputum (색깔, 양)
- [ ] Wheeze
- [ ] Chest pain
- [ ] Fever / Chill
- [ ] Leg edema / 체중 증가
- [ ] Hemoptysis

### 과거력

- [ ] 천식, COPD, 심부전, 폐렴 이력
- [ ] HD(혈액투석) 여부
- [ ] 현재 복용 약물 (기관지확장제, 이뇨제 등)
- [ ] 이전 유사 에피소드 및 입원력

## 감별 진단 포인트

| 진단                   | 주요 특징                                       |
| ---------------------- | ----------------------------------------------- |
| **Acute CHF**          | Orthopnea, PND, leg edema, 최근 체중 증가       |
| **COPD exacerbation**  | 기저 COPD + 가래 증가 + 쌕쌕거림                |
| **Asthma**             | 젊은 연령, 야간 악화, 기관지확장제 반응         |
| **Pulmonary embolism** | Acute onset, pleuritic chest pain, leg swelling |
| **Pneumonia**          | Fever + productive cough + 국소 청진 소견       |
| **Pneumothorax**       | Sudden onset, 한쪽 호흡음 감소                  |

---

## ⚠️ 위험 신호 (즉시 확인)

- **SpO2 < 90%** → 즉각 산소 공급
- **말을 못 할 정도의 숨참** → 중증 호흡부전
- **Tripod positioning / 호흡보조근 사용**
- **Silent chest** (천식 중증 발작)
- **Hypotension + tachycardia** → 긴장성 기흉, 대량 PE
