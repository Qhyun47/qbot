# Abdominal Pain 문진 가이드

## 핵심 문진 항목

### 통증 특성

- [ ] **Location**: 어디가 아픈가? (RUQ / LUQ / RLQ / LLQ / Epigastric / Periumbilical / Diffuse)
- [ ] **Character**: 어떻게 아픈가? (cramping / colicky / constant / sharp / dull)
- [ ] **Onset**: 언제부터? 갑자기 vs 서서히?
- [ ] **Radiation**: 방사통 있는가? (back → pancreatic / right shoulder → cholecystitis)
- [ ] **Severity**: NRS 몇 점? 시간에 따라 변화?

### 소화기 상태

- [ ] **Gas out**: 방귀 나오는가?
- [ ] **Last defecation**: 마지막 배변은?
- [ ] **NPO**: 마지막 식사/음료 섭취 시점?
- [ ] **N/V**: 구역/구토 있는가? 구토 내용물?
- [ ] **Diarrhea / Constipation**

### 동반 증상

- [ ] Fever / Chill
- [ ] Jaundice (황달)
- [ ] Hematochezia / Melena
- [ ] Dysuria / Hematuria (비뇨기과적 감별)

### 여성 추가 문진

- [ ] Last menstrual period (LMP)
- [ ] 임신 가능성 여부
- [ ] 질분비물 / 하복부 압통

### 과거력

- [ ] 이전 복부 수술력 (장폐색 감별 중요)
- [ ] 이전 유사 에피소드 (담석, 맹장염 등)
- [ ] NSAID / 스테로이드 복용 여부

---

## 감별 진단 포인트

| 진단                              | 주요 특징                                             |
| --------------------------------- | ----------------------------------------------------- |
| **Appendicitis**                  | 초기 periumbilical → 이동 to RLQ, McBurney point 압통 |
| **Cholecystitis / Biliary colic** | RUQ colicky pain, 식후(지방식) 악화, Murphy's sign    |
| **Pancreatitis**                  | Epigastric → back radiation, 음주/담석 이력           |
| **Peptic ulcer**                  | Epigastric, 식전/식후 패턴, NSAID 복용                |
| **Intestinal obstruction**        | Cramping + gas out 없음 + 복부 수술력                 |
| **Mesenteric ischemia**           | 고령 + AFib + "pain out of proportion"                |
| **Ectopic pregnancy**             | 여성 + 무월경 + LLQ/RLQ sharp pain                    |

---

## ⚠️ 위험 신호 (즉시 확인)

- **Board-like rigidity / Rebound tenderness** → 복막염
- **Sudden severe onset ("worst pain ever")** → 대동맥류 파열, 장간막 허혈
- **Tachycardia + hypotension** → 복강 내 출혈, 패혈증
- **임신 가능 여성 + 한쪽 하복부 통증** → Ectopic pregnancy rule-out 필수
- **Jaundice + fever + RUQ pain (Charcot's triad)** → Cholangitis
