2. Playwright MCP를 사용해 `http://localhost:3000`을 기준으로 브라우저 동작을 실행합니다.
3. 로그인이 필요한 경우 아래 테스트 계정을 사용합니다:

4. **확인 방법**에 명시된 기준으로 통과 여부를 판정합니다.

판정 기준:

- ✅ **통과**: 오류가 재현되지 않음 (확인 방법의 기대 결과를 충족)

- ⚠️ **확인 불가**: 서버 미실행, 로그인 실패, 항목의 확인 방법이 불명확해 판정 불가 등
